HOW IT WORKS

Three clicks, one running agent

Your cloud account does all the work — you just click

1

Click the cloud button

Hands off to Azure Portal or AWS CloudFormation Console with the OpenClaw template pre-loaded.

2

Pick a model + paste keys

Choose Kimi, Claude, GPT, Gemini, DeepSeek, or Grok. Paste the API key for that provider and your SSH key.

3

Hit Create

Cloud-init runs the bootstrap script. ~60 seconds later your agent gateway is live at the URL shown in the deployment outputs.
