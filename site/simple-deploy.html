Manual Azure Deployment

  1. Download azuredeploy-v2.json
  2. Go to Azure Portal
  3. Search "Deploy a custom template"
  4. Click "Build your own template in the editor"
  5. Paste the JSON content
  6. Click Save, then fill parameters and deploy

Or use Azure CLI:
az deployment group create --resource-group myRG --template-file azuredeploy-v2.json
